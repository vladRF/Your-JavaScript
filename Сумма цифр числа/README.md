# Сумма цифр числа

Вам нужно написать функцию `sumOfDigits(num)`, которая принимает на вход число num, рекурсивно вычисляет сумму его цифр и возвращает результат.

Функция должна корректно работать для любого положительного целого числа (в функцию всегда будет передаваться целое положительное число, либо 0, можно не проверять).

__Пример__
```js
sumOfDigits(123);  // Возвращает 6 (1 + 2 + 3)
sumOfDigits(9876); // Возвращает 30 (9 + 8 + 7 + 6)
sumOfDigits(5);    // Возвращает 5 (только одна цифра)
sumOfDigits(0);    // Возвращает 0
```
                      
                  

__Подсказки__

* Чтобы выделить последнюю цифру числа, используйте оператор % (остаток от деления на 10).
* Чтобы удалить последнюю цифру числа, используйте деление на 10 и округление вниз (например, `Math.floor(num / 10)`).
* Базовый случай рекурсии — когда число равно 0. В этом случае сумма цифр равна 0.
    Убедитесь, что ваша функция корректно работает для числа 0.

__Примечание__

* Ваша функция должна быть реализована именно с использованием рекурсии, без циклов.
* Вызывать функцию не нужно, она будет вызвана автоматически.
