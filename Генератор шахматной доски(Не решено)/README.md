# Генератор шахматной доски

Напишите функцию `generateChessBoard(x)`, которая принимает одно число `x` — размер шахматной доски (количество строк и столбцов). Функция должна возвращать двумерный массив, представляющий шахматную доску размером `x * x`.

Пример возвращаемого значения:
```js
[
  ['.', '#', '.'],
  ['#', '.', '#'],
  ['.', '#', '.'],
]
```

* Белые клетки обозначаются символом `.`.
* Черные клетки на доске обозначаются символом `#`.
* Расположение клеток должно соответствовать стандартному шахматному узору, где левая верхняя клетка (индекс `[0][0]`) всегда белая (обозначается символом `.`).

Примечание

* В функцию всегда будет передаваться "адекватное" число, экстремальных значений не будет, поэтому можно не проверять входящие данные.
* Выводить ничего не нужно, функция должна возвращать значение, сам "рисунок" будет выведен автоматически.
* Функцию вызывать не нужно, она будет вызвана автоматически.
