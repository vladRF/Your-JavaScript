# Волшебная лестница чисел

Вы — строитель магической лестницы. Каждая ступенька этой лестницы состоит из чисел, расположенных в виде треугольника. Чем выше ступенька, тем больше чисел на ней, и они следуют по особым правилам.

Ваша задача — реализовать функцию `printNumberStaircase`, которая построит и выведет в консоль лестницу, используя вложенные циклы. Высота лестницы `(n)` задается параметром функции.

Примечание

* Функция должна вывести лестницу в `console.log()`.
* Обратите внимание, что не должно быть пробелов в конце строки.
* Проверять параметр n не нужно, он всегда будет положительным целым числом.
