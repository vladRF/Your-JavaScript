
# Персонализированное приветствие

Создайте функцию ```greetUser```, которая принимает один аргумент — имя пользователя (строка). Функция должна выводить (через ```console.log```) одно из двух сообщений в зависимости от имени:

1. Если имя — Павел или Олег, то функция должна вывести сообщение:
```"Привет, [имя], вам доступны функции администратора."```

2. Если передано любое другое имя, то функция должна вывести сообщение:
```"Здравствуйте, [имя]."```

__Примечание__

_Вам необходимо лишь реализовать функцию ```greetUser```, вызывать ее не нужно, она будет вызвана автоматически._
